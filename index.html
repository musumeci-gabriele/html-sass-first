Istruzioni di installazione
Node - LavarelMix con Test scrittura Sass

  1. Aprire: Atom e Microsoft Powershell.
  2. Installare su pc Nodesjs tramite il seguente link: https://nodejs.org/it/download
  3. Sul Browser LavarelMix -Stand-Alone Project- tramite link: https://laravel-mix.com/docs/5.0/installation#stand-alone-project
  4. In Atom creiamo la cartella di lavoro
  5. In Powershell posizionarsi nella cartella di lavoro e avviare in ordine:
  6. Nella nostra cartella di lavoro crea una cartella " src " e al suo interno crea questi due file: " app.js "," app.scss "
  7. Su Powershell avvia:
  8. Su Atom seleziona il file package.json e nello scrip inserisci:

    "dev": "npm run development",
    "development": "cross-env NODE_ENV=development node_modules/webpack/bin/webpack.js --progress --hide-modules --config=node_modules/laravel-mix/setup/webpack.config.js",
    "watch": "npm run development -- --watch",
    "hot": "cross-env NODE_ENV=development node_modules/webpack-dev-server/bin/webpack-dev-server.js --inline --hot --config=node_modules/laravel-mix/setup/webpack.config.js",
    "prod": "npm run production",
    "production": "cross-env NODE_ENV=production node_modules/webpack/bin/webpack.js --no-progress --hide-modules --config=node_modules/laravel-mix/setup/webpack.config.js"

  9. Ora in Powershell avvia:
  10. Nota : ricorda di creare il file .gitignore con scritto " node_modules " per evitare il caricamento ni Node si Github.
